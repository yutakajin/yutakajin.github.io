TaskChute Exporter (macOS版)
================================

【概要】
TaskChute Cloud 2のタスクリスト画面とタスク詳細（ノート）を、
自動でPDFとして保存するアプリケーションです。

【使い方】

【インストール・起動方法】

1. インストーラーを開く
   配布された TaskChuteExporter-1.0.0-arm64.dmg をダブルクリックして開きます。

2. アプリのコピー（推奨）
   中にある TaskChuteExporter.app を、Macの「アプリケーション」フォルダなどにドラッグ＆ドロップしてコピーしてください。
   （そのままでも起動できますが、コピーした方が便利です）

3. アプリの起動
   TaskChuteExporter.app をダブルクリックして起動します。
   ※ 初回起動時に「開発元が未確認」という警告が出る場合は、
      アプリを右クリックして「開く」を選択し、ダイアログで「開く」を押してください。
   ※ 入力したURLや保存先フォルダを自動的に記憶し、次回起動時に復元します。

2. ログイン設定 (初回のみ)
   (1) アプリ画面の「1. ログイン設定」にあるボタンを押します。
   (2) ブラウザが立ち上がりますので、TaskChute Cloud 2にログインしてください。
   (3) ログインが完了したら、ブラウザを閉じてください。
       （これでセッション情報が保存されます）

3. スケジュールの登録 / 実行
   (1) 対象URL: TaskChute Cloud 2のタスク画面のURLを入力します。
       (例: https://taskchute.cloud/taskchute?viewId=...)
   (2) 保存先フォルダ: PDFを保存したいフォルダのパスを入力します。
       (例: /Users/username/Documents/Output)
   (3) 「今すぐ実行」ボタンを押すと、即座にPDF生成が行われます。
   (4) 「スケジュールに追加」を押すと、毎日指定した時間に自動実行されます。
       ※ アプリが起動している間のみ、自動実行が動作します。

【出力ファイル】
保存先に以下の2つのファイルが生成されます。
* view_YYYY-MM-DD.pdf : タスクシュートの画面全体のPDF
* tcc2_notes_YYYY-MM-DD.pdf : タスク詳細（ノート）をまとめたPDF

【注意事項】
* アプリのウィンドウを閉じると、終了せずに「通知領域（メニューバー）」に最小化されます。
  （Dockのアイコンは消えますが、アプリは裏で動き続けます）
  ※ これにより、邪魔にならずに自動実行を待機できます。
* 完全に終了するには、メニューバーのチェックマークアイコンをクリックし、「終了」を選択してください。
* Mac (Apple Silicon / M1以降) 専用です。
